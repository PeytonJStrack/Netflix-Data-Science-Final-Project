## Netflix COM-245 Final Project

This project is about taking in data from publically available Netflix data and do analysis on it. I will make plots using the R ggplot2 package as well as use the tidyverse package to filter and orangize the data. Finally, I will be doing some statistical analysis on the data such as a t-tests and chi squared tests.


## How is it made?

Using the R programming language, I imported the netflix_titles.csv file into a dataframe. Then I used the tidyverse package to sort, filter, and orangize the data. Then using the ggplot2 package I create data visualizations from the organized data. This is done 8 different times for 8 different subsets of data. Additionally, I used 8 different kinds of graphs to display proficiency across all kinds of ggplot2 graphs.
